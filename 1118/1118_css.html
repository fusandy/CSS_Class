contain等比例縮放，完整顯示整張圖不裁切，因此假設div的寬高與圖片不同時，會出現留白

cover等比例縮放，填滿整個div區塊不留白，可透過position x與y軸的值調整背景圖片位置

box-sizing決定div的CSS寬高給誰，預設值是content

box sizing:content box
box sizing:border box

圖片要先有origin(起始位置)才能定義clip(裁切)的位置 ; 起始位置預設是padding對齊 ; 裁切位置預設是從border切

background origin:border box
background origin:padding box
background origin:content box
(圖片預設是padding對齊時)background clip:border box
(圖片預設是padding對齊時)background clip:padding box
(圖片預設是padding對齊時)background clip:content box
